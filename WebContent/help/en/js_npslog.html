NpsLog class

NpsLog indicates log record object.
The default log files can be simultaneously output by system defined nps.log and application server terminal, i.e.: the catalina.out files of Tomcat.

Self-defined files will not be output by application server terminal. Self-defined files will be placed under the root directory of current website (even if forcibly use absolute path).

ATTATCH语句使用说明

Attach语句用于读取当前上下文中文章的附件，Attach语句出现位置具有一定限制性：
1)文章模板中，未包含在RS标签体内的附件引用表明是对当前文章的附件列表。
2)页面模板中必须被RS语句包含。
3)多层嵌套的RS语句中，指向离该ATTACH最近的RS结果集，且仅在带Topic标记的RS语句中有效。
4)应使用"att."前缀引用附件标签。注意，att必须小写。

Attach语法

<att type=(t:STRING) rows=(rows:int)>
    --在此写入你的正文内容
</att>

参数名 类型 含义
type STRING 可选。表示附件类型，以附件后缀表示，多个类型以;或空格或tab键或,或|间隔，不区分大小写。 type不设置时表示取所有附件，等同于type="*.*"。

例如，".jpg;.gif"表示匹配jpg和gif格式的图片。

".jpg|.bmp|.gif"

"*.jpg|.bmp|*.gif"

".jpg;.bmp;.gif"

"jpg|bmp|gif"

"jpg bmp gif"

rows int 可选。表示附件取的个数。不填或<=0表示取所有。
和type联合使用时表示取TYPE类型附件的最大条数。

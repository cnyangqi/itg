NpsChineseConverter class

NpsChineseConverter用于汉字的简繁转换，支持大陆、台湾、香港、澳门、新加坡、大马专用词典。注：词典来源于MediaWiki。

Method Summary

String ToSimplified(String text,int synonym) throws Exception
转换为简体中文。text=待转换串,synonym=同义词字典
synonym=1，表示通用简体字典
synonym=3，表示大陆特殊用法，将自动包含通用简体字典
synonym=7，表示新加坡特殊用法，将自动包含通用简体字典
String ToTraditional(String text,int synonym) throws Exception
转换为繁体中文。text=待转换串,synonym=同义词字典
synonym=2，表示通用繁体字典
synonym=4，表示台湾特殊用法，将自动包含通用繁体字典
synonym=5，表示香港特殊用法，将自动包含通用繁体字典
synonym=6，表示澳门特殊用法，将自动包含通用繁体字典
synonym=8，表示大马特殊用法，将自动包含通用繁体字典

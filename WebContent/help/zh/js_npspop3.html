NpsPop3 class

NpsPop3用于接收电子邮件。

Method Summary

NpsPop3 NpsPop3(String host,int port)
构造函数
void Connect(String user,String passwd)
使用给定的邮件帐号进行POP3认证。
void ConnectSSL(String user,String passwd)
使用SSL连接进行POP3认证。
void PostTo(NpsTopic topic,String prefix_subject,boolean delete_mail)
收取未读邮件，并提交到指定的栏目。
如果需要根据主题前缀进行过滤，请设置prefix_subject， 例如prefix_subject="news:"表示以news:开头的所有邮件；prefix_subject=null或""表示不进行过滤。
delete_mail表示正常提交后是否删除远程邮箱中的邮件。true表示完成后自动删除邮件。
void Close()
关闭连接，并更新邮箱。

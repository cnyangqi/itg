MAP语句使用说明

MAP语句用于读取栏目列表，MAP语句内可以嵌套MAP语句、RS语句和JAVA语句。

MAP语法

<map id=(id:STRING) site=(site:STRING) topic=(topic:STRING)>
    --在此写入你的正文内容
</map>

参数名 类型 含义
id STRING 必须。该map的唯一号。必须以字母或下划线开头。
site STRING 可选。要查询的站点代号。
可以使用各类标签。
topic STRING 可选。要查询的栏目代号。
可以使用各类标签。

MAP内标签使用说明

1.以<$开头，$>结尾定义标签，标签必须以map语句中的id作为前缀，例如<$map1.top_name$>。
2.名称以top_开头的全局标签有效，含义同全局标签。具体有：

标签名 含义
top_id 栏目唯一标识号
top_name 栏目名称
top_alias 栏目别名
top_code 栏目代号
top_url 栏目URL访问路径
top_path 同top_url
top_navigator 当前栏目所在位置，使用>进行间隔
top_nav 同top_navigator
top_parentid 父栏目id号
top_parentcode 父栏目代号
top_parentname 父栏目名称
top_parentalias 父栏目别名
top_parenturl 父栏目url路径
top_parentpath 同top_parenturl


样例代码


以下代码表示读取当前模板所在站点的顶级栏目信息。
<map id="map1">
     <$map1.top_name$>
</map>

以下代码表示读取指定mysite站点的顶级栏目信息。
<map id="map1" site="mysite">
     <$map1.top_name$>
</map>

以下代码表示读取当前模板所在站点，栏目代号为top1的下级栏目列表。
<map id="map1" topic="top1">
     <$map1.top_name$>
</map>

以下代码表示读取当前模板所在栏目的下级栏目列表。
<map id="map1" topic=<$top_code$>>
     <$map1.top_name$>
</map>

以下代码表示读取站点mysite下栏目代号为top1的栏目的下级栏目列表。
<map id="map1" site="mysite" topic="top1">
     <$map1.top_name$>
</map>

以下代码同样表示读取站点mysite下栏目代号为top1的栏目的下级栏目列表。
<map id="map1" topic="mysite.top1">
     <$map1.top_name$>
</map>

以下代码与以上代码不同，仅表示读取站点mysite下栏目代号为mysite.top1的栏目的下级栏目列表。
<map id="map1" site="mysite" topic="mysite.top1">
     <$map1.top_name$>
</map>

NpsTopic class

NpsTopic表示栏目对象。

Property Summary

String id
栏目唯一标识。
String name
栏目名称。
String alias
栏目别名。
String code
栏目代号。
int default_state
默认文章录入后状态。
0:草稿，1:提交待审核，2:审核通过待发布，3:已发布
float score
文章默认分值。
String table
自定义数据源表名
NpsTopic parent
上级栏目。null表示顶级栏目。
String keywordlinks
热字。一行是一组热字。一组热字中，多个关键字必须以逗号(,)间隔，最后一个是URL链接。
boolean solr_enabled
是否启用solr。
String solrcore
solr core.
boolean sort_enabled
是否支持混排。

Method Summary

boolean IsNormal()
是否为通用新闻栏目。true:是，false:不是
boolean IsCustom()
是否为自定义数据源。true:是，false:不是
void Build(boolean rebuildall)
重建栏目，栏目页面模板默认按自底向上进行重建（效果同BuildPagesOnly）。
rebuildall=true，表明重建整个栏目。
rebuildall=false，表明发布待发布的文章。
void BuildPagesOnly()
按自底向上重建栏目页面模板。
void BuildPagesTopDown()
按自顶向下重建栏目页面模板。
int Sync()
同步自定义数据源数据，并发布
void Archive()
按栏目提供的归档方案归档。
void AddKeywordLink(String keyword,String url)
增加热字
void AddKeywordLinks(String line)
增加一组热字。多个关键以逗号(,)间隔，最后一个是链接。
void RemoveKeywordLink(String keyword,String url)
删除热字
void RemoveKeywordLinks(String url)
删除链接到url的一组热字

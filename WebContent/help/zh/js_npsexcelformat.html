NpsExcelFormat class

NpsExcelFormat用于格式化Excel单元格样式。

样例代码：
var format1 = new NpsExcelFormat("","");
format1.SetFont("宋体",12,true,false,false,"black");
format1.SetAlignment("centre");
format1.SetVerticalAlignment("centre");
format1.SetBorder("all","thin");
format1.SetBackground("yellow");
format1.SetOrientation("vertical");

Property Summary

Method Summary

NpsExcelFormat new(String type,String format_str)
构造函数。type表示format_str的类型：
1.type=date，表示format_str是格式日期参数；
2.type=number，表示format_str是格式数字参数；
void SetFont(String font_name,int point_size,boolean bold,boolean italic,boolean underline,String color)
设置字体。
font_name:字体类型，例如宋体、楷体、Arial等等；
point_size:字体大小；
bold=true表示粗体；
italic=true表示斜体；
underline=true表示带下划线；
color是字体颜色，目前仅支持red,blue,yellow,green,grey,white,black；
void SetAlignment(String align)
设置水平对齐方式。可以有以下取值：
left：居左；
right：居右；
centre：居中；
justify：两端对齐；
fill： 缩小字体填充
void SetVerticalAlignment(String align)
设置垂直对齐方式。可以有以下取值：
top：靠上；
bottom：靠下；
centre：居中；
justify：两端对齐；
void SetBorder(String border,String style)
设置单元格边框样式。
border表示是否有边框，有以下取值：
none:无
all:外边框
top:上
bottom:下
left:左
right:右

style表示边框样式，有以下取值：
none
thin
thick
dash_dot
dash_dot_dot
dashed
dotted
double
hair
medium
medium_dash_dot
medium_dash_dot_dot
medium_dashed
SLANTED_DASH_DOT
void SetBackground(String color)
设置单元格背景色。目前仅支持red,blue,yellow,green,grey,white,black。
void SetIndentation(int i)
设置缩进。
void SetLocked(boolean b)
设置是否加锁。
void SetOrientation(String orientation)
设置文字方向。
vertical:垂直；
horizontal:水平；
void SetShrinkToFit(boolean b)
设置是否缩小字体填充。
void SetWrap(boolean wrap)
设置自动换行。

工作流节点可以使用JavaScript、DynamicJava、SQL三种语言进行业务编程。这里着重介绍如何使用DynamicJava进行二次开发。

全局变量

变量名 类型 含义
session WorkflowSession 记录了当前上下文情景
out NpsLog 日志写入全局文件nps.log
user User 记录当前运行账户
app_id String 申请表ID#
app_title String 申请表标题
app_emergency int 紧急程度。 0普通 5紧急 9特急
app_businesstype BusinessType 业务类型
app_process WorkProcess 业务流程
app IApp 业务数据
action Action 操作，读取操作类型、审批意见、下一环节审批人等信息

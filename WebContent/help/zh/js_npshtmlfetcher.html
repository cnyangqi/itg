NpsHtmlFetcher class

NpsHtmlFetcher用于HTML网页的抓取、保存和图片的缩放。

Method Summary

void AddNameValuePair(String name,String value)
为Login设置参数，name为字段名，value为其对应值。例如：
AddNameValuePair("username","system");
AddNameValuePair("password","manager");
void Login(String url)
登录指定站点。
NpsHtmlParser GetHtmlParser(String url,String encoding) throws Exception
获取url指定的网页，并对该网页字段进行分析获得NpsHtmlParser对象
void SaveHtml2File(String url,NpsFile output) throws IOException
获取url指定网页并另存到output文件中
void GetBinary(String url,NpsFile output) throws IOException
获取url指定的二进制文件（例如图片）并另存到output文件中
void ScaleTo(NpsFile src,NpsFile dest,int width,int height) throws Exception
将src文件中的图片按width和height进行缩放，写入dest文件中
width=-1表示按高度自动缩放；height=-1表示按宽度自动缩放
